# SDKWork GitHub

GitHub integration application for repository management, issues, feature checklists, and planning.

## Layout

- `apps/sdkwork-github-pc/` — PC React application surface
- `apis/` — OpenAPI contract authorities
- `crates/` — Rust API server, route crates, services, repositories
- `database/` — `sdkwork-database` lifecycle assets
- `sdks/` — SDK families and route manifests

Standards: `../sdkwork-specs/README.md`

## Commands

```bash
pnpm dev
pnpm check
pnpm verify
```
